    • [2018] Project Support Procedures

    The following table details the labels assigned to trouble tickets for support and general maintenance of the "begin_2_nvy" project. Any questions can be posted as trouble tickets in the GitHub project repo.

    Issues | Labels & Assignments

    •   A | Add [ #00ff90 ]
      • Add usability or functionality
    •   A | Add [ Documentation ]
      • Add supporting documentation for the project
    •   A | Add [ Feature ]
      • Add an enhancement or feature to support user experience
    •   C | Change [ #62f8ed ]
      • Valid changes to the product and committed code.
    •   C | Change [ Bug ]
      • Changes improve or eliminate bugs
    •   C | Change [ General ]
      • Valid changes to the product and committed code, includes superficial alterations
    •   M | Maintenance [ #bfbfbf ]
      • Changes required for the product or underlying infrastructure and security of supporting components
    •   M | Maintenance [Security ]
      • Changes made for the security of the product and community
    •   D | Delete [ #ff006fab ]
      • Removal of functionality, function or feature of the project Use sparringly
    •   UNK | NEW [ #b60205 ]
      • Issue unknown, not yet identified or confirmed. Will be auto-assigned to all new issues for resolution.
    •   RFI | Info Req [ #d93f0b ]
      • Request for information on the product or an aspect of product or its usage
